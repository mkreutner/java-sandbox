# Vérification de l'unicité des éléments dans un tableau

Écrivez un programme Java qui vérifie si tous les éléments d'un tableau sont uniques, 
c'est-à-dire qu'il n'y a pas de doublons dans le tableau. 

L'utilisateur doit saisir la taille du tableau et les éléments du tableau, et le 
programme doit déterminer si tous les éléments sont uniques ou non. 

Vous devez coder la logique de vérification de l'unicité des éléments en Java et vous 
assurer que votre programme fonctionne correctement. 

Vous pouvez également ajouter des commentaires pour expliquer votre code.
